========

[![Dependency Status](https://david-dm.org/wuhy/fisx-command-search.svg)](https://david-dm.org/wuhy/fisx-command-search) [![devDependency Status](https://david-dm.org/wuhy/fisx-command-search/dev-status.svg)](https://david-dm.org/wuhy/fisx-command-search#info=devDependencies) [![NPM Version](https://img.shields.io/npm/v/fisx-command-search.svg?style=flat)](https://npmjs.org/package/fisx-command-search)

> Search component package information for fisx.

## how to use

### search component package

```shell
fisx search er # search package from default endpoint, by default `edp`
fisx search github:er # search package from github, more search option see the help information
# fisx search npm:jquery # need a long time to init, it's not suggested to using it now!
```

### view help information
    
```shell
fisx search -h
```
